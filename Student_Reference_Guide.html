🚀 Deploy Your Learning Portal on Netlify

How to Deploy This Folder:

  1. Go to netlify.com and sign up (free account)
  2. Download and install Git on your computer
  3. Install Node.js (includes npm)
  4. Open terminal/command prompt in this folder and run: npm install -g netlify-cli
  5. Run: netlify deploy --prod
  6. Follow the prompts and link to your GitHub account
  7. Your site will be live! Share the Netlify link with students

✅ Once deployed, all files will be accessible from the Netlify URL

Tip: Every time you update this folder, run netlify deploy --prod again to push updates to your live site.
