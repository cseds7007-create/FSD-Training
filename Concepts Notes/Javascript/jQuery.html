1. What is DOM?

DOM stands for Document Object Model. Browser converts HTML page into tree structure. JavaScript can access and manipulate HTML elements using DOM.

DOM Tree Structure

HTML
└── BODY
    ├── H1
    ├── P
    └── BUTTON

Why DOM is Important?

  • Change webpage dynamically
  • Handle user interactions
  • Create animations
  • Update content without refresh
  • Used heavily in modern frameworks

2. Introduction to jQuery

jQuery is a lightweight JavaScript library that simplifies DOM operations.

Without jQuery With jQuery
Long JavaScript code Short and easy code
Browser compatibility issues Handled automatically

Add jQuery

<script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>

Basic Syntax

$(selector).action()

3. Selecting Elements

Selectors are used to access HTML elements.

Selector Description
$("p") Select all paragraphs
$("#id") Select by ID
$(".class") Select by Class
jQuery
$("#demo").css(
 "background","yellow"
);
Normal JavaScript
document.getElementById("demo")
.style.background = "yellow";

This paragraph will be highlighted.

4. Manipulating Styles

Styles can be changed dynamically using jQuery and JavaScript.

jQuery
$("#box").css({
 background:"black",
 color:"white",
 padding:"20px"
});
Normal JavaScript
let box =
document.getElementById("box");

box.style.background = "black";
box.style.color = "white";
box.style.padding = "20px";
Style changing demo

5. Manipulating Text

Method Purpose
text() Change plain text
html() Change HTML
val() Get input value
jQuery
let name = $("#name").val();

$("#output").text(name);
Normal JavaScript
let name =
document.getElementById("name")
.value;

document.getElementById("output")
.innerText = name;

6. Adding & Removing Elements

Method Purpose
append() Add element at end
prepend() Add element at start
remove() Remove element
jQuery
$("#list").append(
 "<li>New Item</li>"
);
Normal JavaScript
let li =
document.createElement("li");

li.innerText = "New Item";

document.getElementById("list")
.appendChild(li);

8. Event Handling

Event Description

keydown() Keyboard key press
jQuery

 alert("Clicked");
});
Normal JavaScript
document.getElementById("btn")
.addEventListener("click",

 alert("Clicked");
