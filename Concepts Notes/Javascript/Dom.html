How HTML Parsing Works?

The browser reads your HTML code and converts it into a DOM (Document Object Model). JavaScript then accesses this DOM and manipulates the webpage dynamically.

Document
Root Element <html>
Element <head>
<title>
<meta>
Element <body>
<h1>
<p>

Structure

DOM represents HTML as a tree structure.

Manipulation

JavaScript can add, remove, or modify elements.

Dynamic

Changes in DOM are reflected instantly.

1. HTML Source Code


<body>

    <h1 id="title">
        Hello World
    </h1>

    <p class="text">
        My Paragraph
    </p>

</body>

2. JavaScript Access 


// Accessing by ID

document.getElementById("title");


// Accessing by Class

document.getElementsByClassName("text");


// Query Selector

document.querySelector("p");
