JavaScript Variables & Scope Demonstration

1. Why Variables?

Variables are containers used to store data.

Example: let age = 22;
Here:
  • age → variable name
  • 22 → stored value

2. var vs let vs const

Keyword Can Reassign? Scope Modern Usage
var YES Function Scope Avoid Mostly
let YES Block Scope Most Common
const NO Block Scope Best for Fixed Values

3. Understanding var

Problem with var:
  • Function scoped
  • Can redeclare variables
  • Can create bugs

4. Understanding let

Why let is better:
  • Block scoped
  • Safer than var
  • Can update values

5. Understanding const

Why use const:
  • Prevents accidental reassignment
  • Makes code safer
  • Used for fixed values

8. Where Should We Use Them?

Use const

Default choice.

Use when value should NOT change.


const PI = 3.14;

Use let

Use when value WILL change.


let score = 0;

Avoid var

Old JavaScript way.

Can cause scope confusion.


var x = 10;
