JavaScript Promises

Learn Promises, resolve(), reject(), then(), catch(), async/await, callbacks, and asynchronous JavaScript visually.

What is a Promise?

A Promise is a built-in JavaScript object used for handling asynchronous operations.


Real Life Analogy

  • You order food online
  • The food is not delivered immediately
  • The app promises delivery later
  • Delivered → Success
  • Cancelled → Failure

Promise States

Pending → Work is still happening
Fulfilled → Operation completed successfully
Rejected → Operation failed

Your Promise Example

let promise = new Promise((resolve,reject)=>{

    let success = true;

    if(success){
        resolve("Data Loaded");
    }else{
        reject("Error");
    }

});

resolve() and reject()

JavaScript automatically provides these functions.


resolve() → Marks promise as success
reject() → Marks promise as failure

Execution Flow

Promise Created ↓ Pending ↓ success ? / \ yes no ↓ ↓ resolve reject ↓ ↓ fulfilled rejected

then() and catch()

then()

.then(data => console.log(data))
  • Runs when promise succeeds
  • Receives value from resolve()
  • Handles successful operations

catch()

.catch(err => console.log(err))
  • Runs when promise fails
  • Receives value from reject()
  • Handles errors

Promise Methods

Method Purpose
then() Handle successful operations
catch() Handle errors
finally() Runs always
Promise.all() Waits for all promises
Promise.race() Returns first completed promise
