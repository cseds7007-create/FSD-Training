ES6 + DOM Manipulation Guide

1. let & const

Concept

ES6 introduced let and const to solve issues caused by var.

  • let → value can change
  • const → cannot reassign
  • Both are block scoped
Feature var let const
Scope Function Block Block
Reassign Yes Yes No

Interactive Example

let age = 20; age = 25; console.log(age);

2. Arrow Functions

Concept

Arrow functions provide shorter syntax for functions.

const add = (a,b) => a + b;

Arrow functions do NOT create their own this.

3. Spread Operator

Concept

Spread operator expands array or object elements.

const nums = [1,2,3]; const copy = [...nums];

4. Rest Operator

Concept

Rest operator collects multiple values into an array.

function total(...nums){ return nums.reduce((a,b)=>a+b); }
