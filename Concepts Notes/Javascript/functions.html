2. Parameters vs Arguments

Parameter = Placeholder
➡
Argument = Actual Value
function add(a,b){ // parameters return a+b; } add(5,10); // arguments

3. Return Statement

Return sends data back from the function. After return executes, function stops.

function multiply(a,b){ return a*b; } let result = multiply(5,4); console.log(result);

4. Callback Functions

What Problem Do Callbacks Solve?

Sometimes JavaScript should wait until another task finishes. Example:

  • Button Click
  • API Response
  • Timer Completion
  • File Loading

"Run this function later when task finishes"

Main Function
➡
Receives Callback
➡
Executes Later
function processUser(name, callback){ console.log("Processing user..."); callback(name); } function greetUser(name){ console.log("Hello " + name); } processUser("Krishna", greetUser);

5. Why JavaScript Needs Async?

JavaScript is Single Threaded. It runs one line at a time.

If one task takes long time:

Entire webpage freezes.

Solution → Async Programming

7. Promises

Promises represent future completion of an async operation.

Pending

Fulfilled

let promise = new Promise((resolve,reject)=>{ let success = true; if(success){ resolve("Data Loaded"); }else{ reject("Error"); } }); promise .then(data=>console.log(data)) .catch(err=>console.log(err));

8. Why Async/Await?

Problem With Promises

Too many .then() chains become difficult.

fetchData() .then() .then() .then()

Solution → async / await

Looks like synchronous code. Cleaner and easier to read.

async function getData(){ let result = await fetchSomething(); console.log(result); }
await

9. Higher Order Functions

A Higher Order Function:

  • Takes another function as argument
  • OR returns another function

Why Use Them?

  • Reusable Logic
  • Cleaner Code
  • Functional Programming
  • Used heavily in React
function calculate(a,b,operation){ return operation(a,b); } function add(x,y){ return x+y; } console.log(calculate(5,3,add));

10. map(), filter(), reduce()

map()

Transforms every element.

let nums = [1,2,3]; let doubled = nums.map(n=>n*2);

filter()

Filters elements.

let even = nums.filter(n=>n%2===0);

reduce()

Accumulates values.

let total = nums.reduce((sum,n)=>sum+n,0);
