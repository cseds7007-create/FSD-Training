The Cascade - Specificity and Inheritance

  1. Specificity

    Specificity is a set of rules that browsers use to determine which CSS rule applies when multiple rules could apply to the same element. It is calculated based on the types of selectors used in the CSS rule.

    The specificity hierarchy is as follows:

    • Inline styles (e.g., style="color: red;") have the highest specificity.
    • ID selectors (e.g., #header) have a higher specificity than class selectors.
    • Class selectors (e.g., .nav) have a higher specificity than element selectors.
    • Element selectors (e.g., div, p) have the lowest specificity.

  2. Inheritance

    Inheritance is a mechanism by which some CSS properties are passed down from parent elements to their children. Not all properties are inherited; for example, text-related properties like color and font-family are typically inherited, while box model properties like margin and padding are not.

    Understanding which properties are inherited can help you write more efficient CSS and avoid unnecessary declarations.
