Bootstrap Linking And Usage Demo

How to link Bootstrap in an HTML page and How Bootstrap classes can be used for modern responsive styling.

Step 1 : Linking Bootstrap In HTML

Bootstrap CDN Link

Add the Bootstrap CDN link inside the head section of the HTML page. 

<head>

    <link 
        href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" 
        rel="stylesheet"
    >

</head>

Why Use CDN?

  • Faster Bootstrap setup
  • No manual file download required
  • Easy to use in projects
  • Automatic responsive support

Step 2 : Using Bootstrap Classes

Buttons

Bootstrap provides ready-made button styles. 

<button class="btn btn-primary">
    Primary
</button>

Alerts

Bootstrap alerts are useful for notifications.

Warning Alert Example 
<div class="alert alert-warning">
    Warning Alert
</div>

Cards

Cards help organize content beautifully.

Simple Bootstrap Card 
<div class="card">

    <div class="card-body">
        Card Content
    </div>

</div>
